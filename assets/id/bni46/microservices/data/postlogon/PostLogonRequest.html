public final class PostLogonRequest

    PostLogonRequest adalah data class yang merepresentasikan permintaan untuk operasi setelah proses logon. Kelas ini menyimpan informasi seperti ID instance (iid), Merchant MID (mMID), Terminal TID (mTID), dan nomor serial (serialNo) yang diperlukan untuk operasi tersebut.
